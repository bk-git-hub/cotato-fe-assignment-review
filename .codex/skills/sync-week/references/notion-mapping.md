# Notion Mapping

Map the week page into review calibration like this:

## 읽을 거리

Use this to infer concept scope.

This mainly informs:
- `Optional Review Comments`
- `Skip Comments For This Week`

## 실습

Use this as the main review target.

This mainly informs:
- `Required Review Focus`

## 제출 기준 / Done

Use this to sharpen what matters most in review.

Typical examples:
- visual similarity to Figma
- spacing, typography, color, radius, shadow
- component split
- import/export clarity

## Critical Issues Override

Always add this section explicitly even if the week page does not mention it.

This section exists so `review-pr` will still surface important issues outside normal pedagogy scope.
